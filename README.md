# 墨芯 v5.0 - 小说技法智能分析系统

基于规则匹配的小说写作技法提取与分析系统。

## 功能特性

- **技法提取**: 自动识别小说中的写作技法
- **全本分析**: 支持整本小说的批量处理
- **多维分类**: 7大类别（情节、人物、场景、对话、情感、节奏、主题）
- **置信度评分**: 每个技法附带置信度评估
- **对比分析**: 支持多部作品的技法对比

## 快速开始

```bash
# 提取单部小说技法
python3 extract_techniques.py 间客 1 50

# 批量提取
./batch_extract.sh 间客 50
```


## 技法分类

- **plot**: 情节类（悬念设置、危机开场、高潮爆发等）
- **character**: 人物类（人物出场、内心独白、群像描写等）
- **scene**: 场景类（环境描写、空间转换、氛围营造等）
- **dialogue**: 对话类（对话推进、潜台词等）
- **emotion**: 情感类（情感铺垫、情绪爆发、温情描写等）
- **pacing**: 节奏类（快节奏、慢节奏、场景切换等）
- **theme**: 主题类（哲学思辨、社会隐喻等）
